# SandboxJS

## Prerequesites

* NodeJS
* NPM
* Gulp

## NPM Modules
```bash
$ npm install -g webpack
$ npm install -g less
$ npm install -g coffee-script
```

## Launch the application

```bash
$ npm install # downloads all JS dependencies
$ webpack # pre-processes static resources
```

Then, in a browser, open the file ```./index.html```.
